| 控制符 | 说明                                |
| ------ | ----------------------------------- |
| \r     | 回车，将当前位置移到本行开头        |
| \n     | 换行，将当前位置移到下一行          |
| \t     | 水平制表（HT）（跳到下一个TAB位置） |
| \ \    | 代表一个反斜线字符                  |
| \ '    | 代表一个单引号字符                  |
| \ "    | 代表一个双引号字符                  |
| \?     | 代表一个问号                        |
| \0     | 空字符（NUL）                       |
