| 控制符 | 说明                                                         |
| ------ | ------------------------------------------------------------ |
| %d     | 按十进制整型数据的实际长度输出                               |
| %ld    | 输出长整型数据                                               |
| %md    | m为指定的输出字段的宽度。如果数据的位数<m，则左端补以空格，如果>m，则按实际位数输出 |
| %c     | 输出字符                                                     |
| %s     | 输出字符串                                                   |
| %u     | 输出无符号整型（unsigned）                                   |
| %f     | 输出实数，包括单精度和双精度，以小数形式输出                 |
| %x     | 以十六进制形式输出整数                                       |
